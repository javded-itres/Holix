# Licensing strategy

Holix core is currently distributed under the **MIT License** (`LICENSE`, `pyproject.toml`).

The full strategic analysis (Open Core, dual AGPL, enterprise boundary) is maintained in Russian:

**[../ru/LICENSING_STRATEGY.md](../ru/LICENSING_STRATEGY.md)**

Summary:

| Component | Current / planned |
|-----------|-------------------|
| Agent runtime, CLI, TUI, self-hosted gateway | Open source (MIT / Apache 2.0 under discussion) |
| Multi-tenant control plane, billing, SSO, audit | Planned commercial layer (Open Core) |
| Managed cloud (holix-agent.ru) | Service + SLA |

This document is strategic, not legal advice. Consult counsel before changing the license.
